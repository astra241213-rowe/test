# 今晚操作指南（Mac 版）

## 第一步：把项目放进你的 git 仓库
解压后，把整个文件夹里的文件拖进你之前建好的本地 git 仓库文件夹，然后在终端（Terminal）里：

```bash
cd 你的仓库文件夹路径        # 提示：输入 cd 加空格后，直接把文件夹拖进终端窗口就会自动填路径
git add .
git commit -m "init: 项目骨架"
git push
```

## 第二步：安装依赖
```bash
pip3 install -r requirements.txt
```
如果报错 `command not found: pip3`，先运行 `python3 -m ensurepip`，或告诉 Claude 报错内容。

## 第三步：跑起来
```bash
streamlit run app.py
```
浏览器会自动打开 http://localhost:8501 —— 现在是 DEMO 模式（离线假数据 + 预置文案），
**不需要任何 API key 就能看到完整界面和流程**。今晚的目标就是跑通这个。
